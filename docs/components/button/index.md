# Button

Button 是用户界面中承载交互行为的基础元素，通过点击或触摸触发特定功能（如提交表单、导航页面、执行操作等）。它是连接用户意图与系统响应的桥梁，设计需兼顾*可识别性*、*可操作性*和*视觉引导性*。
