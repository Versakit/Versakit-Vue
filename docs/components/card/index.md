# Card

Card 是一种将相关信息聚合在一起的容器组件，通常以矩形卡片形式呈现，具有独立的视觉边界。它通过分组内容创造清晰的视觉层次，支持多样化的内容结构（如图片、文本、按钮），广泛应用于信息展示、内容聚合和操作入口场景。
