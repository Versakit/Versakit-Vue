# Popover

Popover 是一种轻量级的浮动层组件，用于在触发元素（如按钮、图标）附近显示额外信息或操作选项。它通常以箭头指向触发元素，支持多种触发方式（点击、悬停、聚焦），在不占用主视图空间的前提下提供上下文相关内容。
