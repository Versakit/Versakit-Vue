# Tooltip

Tooltip 是一种轻量级信息提示组件，当用户悬停、聚焦或点击元素时，会在其附近显示简短说明文本。它以半透明气泡形式呈现，不干扰主界面，用于提供即时的上下文帮助或补充信息，常见于图标解释、功能说明等场景。
