# Carousel 轮播组件

Carousel 是一个内容轮播展示组件，支持自动轮播、手动切换、触摸滑动等交互方式。它可以用于展示一组图片、卡片或其他任何内容，并提供水平和垂直两种方向的轮播方式，以及滑动和淡入淡出两种过渡效果。
