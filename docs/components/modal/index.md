# Model

Modal 是一种覆盖在主界面上方的浮动容器，用于展示需要用户专注处理的内容（如表单、确认框、详情页等）。它通过半透明背景（遮罩层）阻断主界面交互，强制用户完成当前操作后才能返回，是现代 UI 设计中实现层级交互的重要组件。
