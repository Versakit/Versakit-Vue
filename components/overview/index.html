组件总览

Versakit 提供了一系列功能丰富、易于使用的组件，用于构建现代化的用户界面。

A

Alert

用于显示重要信息或操作结果的通知组件。

A

Avatar

表示用户或对象身份的图形化标识，通常显示为圆形或方形的图片、图标或字母组合。

B

Badge

用于显示状态标记或通知计数的小型视觉指示器，通常以圆形或椭圆形徽章形式出现。

B

Button

用于触发用户操作的交互元素，通常以按钮形式呈现。

C

Card

用于显示内容和信息的容器组件，通常以卡片形式呈现。

C

Checkbox

用于从多个选项中选择零个或多个条目的交互控件。

C

Chip

用于显示状态标记或通知计数的小型视觉指示器，通常以圆形或椭圆形徽章形式出现。

D

Divider

用于分隔内容区域的视觉元素，通常以线条或空白区域形式呈现。

D

Drawer

常见的移动端和桌面端交互组件，通常以侧边滑出或底部弹出的形式呈现。

I

Input

用于输入单行或多行文本的交互组件，通常以文本框形式呈现。

K

Kbd

用于标识用户需要通过键盘输入的内容（如快捷键、命令行输入等）。

M

Modal

模态框，用于在当前页面层级之上展示重要信息、请求用户确认或执行关键操作。

P

PinInput

专为输入短序列密码（如 PIN 码、验证码）设计的交互组件。

P

Popover

轻量级的浮动层组件，用于在触发元素附近显示额外信息或操作选项。

R

Rate

用于显示评分或评价的交互组件，通常以星星或心形等形式呈现。

S

Segmented

水平排列的选项卡控件，用于在多个互斥选项之间进行快速切换。

S

Select

用于从预定义选项列表中选择单个或多个条目的交互控件。

S

Slider

允许用户通过拖动滑块在指定范围内选择连续或离散值的交互控件。

S

Switch

二元状态选择器，用于在两种状态间快速切换（如 "开 / 关"、"启用 / 禁用"）。

T

Tabs

常见的界面模式，用于在有限空间内组织和切换多个内容面板。

T

Textarea

用于输入多行文本的交互组件，通常以文本框形式呈现。

T

Tooltip

轻量级信息提示组件，当用户悬停、聚焦或点击元素时，会在其附近显示简短说明文本。
